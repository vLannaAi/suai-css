SUAI (สวย)

A progressive-enhancement CSS framework in four stages. Pick a Skin and Mode above — every demo below re-skins live from the same --su-* token contract.

Stage 1 — Semantic HTML, zero JavaScript

Plain semantic HTML styled by a single generated suai.bundle.css. No build tools and no JavaScript required to render; the Skin / Mode switcher is an optional progressive enhancement.

  1. Text elements
  2. Display elements
  3. Form controls
  4. Media elements
  5. Layout elements
  6. Theme reference

Stage 2 — Token contract (UnoCSS)

The canonical --su-* token contract consumed as UnoCSS utility classes that compile to var(--su-*), so compiled component CSS re-skins at runtime with no rebuild.
