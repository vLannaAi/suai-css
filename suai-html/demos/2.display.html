Details/Summary - Simple Accordion

What is SUAI?

SUAI (Semantic Utility Adaptive Interface) is a CSS framework designed for the AI era, focusing on semantic HTML and visual-only UX. It provides beautiful styling for standard HTML elements without requiring utility classes.

How does SUAI differ from other CSS frameworks?

Unlike utility-first frameworks like Tailwind CSS, SUAI styles HTML elements directly based on their semantic meaning. This approach is ideal for AI-generated content and reduces the complexity of maintaining countless utility classes.

What themes are available?

SUAI currently offers two themes:

  • Pla (Fish) - Material Design 3 inspired with blue primary color
  • Nok (Bird) - Bootstrap inspired with traditional styling

Each theme supports both light and dark modes.

Details/Summary - Nested Accordion

Advanced Features

SUAI includes several advanced features for modern web development:

Theme System

Switch between Pla and Nok themes at runtime using the SUAI JavaScript API. Themes can be changed without page reload, and preferences are stored in localStorage.

Typography

Uses Roboto Flex variable font with automatic optical sizing. The slant axis (-10°) is used for italic elements instead of fake oblique styles, providing better typography.

Color System

Material Design 3 color palette with semantic colors:

  • Primary: #3b82f6 (MD3 Blue)
  • Success: Green
  • Warning: Orange
  • Danger: Red
  • Info: Light Blue
PWA Support

All fonts and assets are available locally, making SUAI perfect for Progressive Web Apps that need offline functionality.

Datalist - Autocomplete Suggestions

Note: The <datalist> provides autocomplete suggestions. Browser implementation varies - some show a dropdown, others filter as you type.

Inline Definitions

A Progressive Web App (PWA) is a type of application software delivered through the web, built using common web technologies including HTML, CSS, and JavaScript.

The DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

Semantic HTML refers to the use of HTML markup to reinforce the semantics, or meaning, of the information in web pages rather than merely to define its presentation or look.

Technical Glossary

CSS Custom Properties
Also known as CSS variables, these are entities defined by CSS authors that contain specific values to be reused throughout a document. They are set using custom property notation (e.g., --main-color: black;) and accessed using the var() function.
Variable Font
A single font file that behaves like multiple fonts. OpenType variable fonts allow different variations of a typeface to be incorporated into one file, with axes like weight, width, slant, and optical size controllable via CSS.
Material Design
A design language developed by Google in 2014. Material Design 3 is the latest version, featuring dynamic color, improved accessibility, and refined components based on Material You design principles.
Semantic Versioning
A versioning scheme for software that uses a three-part number: MAJOR.MINOR.PATCH. Increment the MAJOR version for incompatible API changes, MINOR for backwards-compatible functionality, and PATCH for backwards-compatible bug fixes.

About SUAI Themes & Modes

  • Mode: light
  • No JavaScript required for styling
  • Tag-based styling - no classes needed for basic elements

Production Setup

Link the single bundled stylesheet; every theme and mode is included, selected via data-su-theme / data-su-mode attributes:

<link rel="stylesheet" href="suai-html/suai.bundle.css">

<html data-su-theme="nok" data-su-mode="dark">

Interactive Theme Switching

For demo/interactive switching, add the optional switcher script (not required for production):

<script src="suai-html/suai-theme.js" defer></script>
<button data-su-set-theme="pla">PLA</button>
<button data-su-set-mode="dark">Dark</button>
