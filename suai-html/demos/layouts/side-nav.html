Dashboard content

An application shell: header across the top, a navigation sidebar on the left, the main content area, and a footer across the bottom — placed with CSS Grid.

On mobile the sidebar is a hidden drawer; tap Menu to slide it in and Close to dismiss it — no JavaScript (a hidden checkbox toggles it, so it never touches the URL or scroll position). On tablet and desktop (≥768px) the sidebar is permanent.

When to use

  • Admin panels and dashboards
  • Apps with a persistent primary navigation
