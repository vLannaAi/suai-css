Content

A mobile-app shell. On mobile the navigation is a fixed tab bar pinned to the bottom of the screen, with the content scrolling above it. On tablet and desktop (≥768px) the same nav becomes a normal top bar in the document flow.

The main area reserves space at the bottom on mobile so the fixed bar never covers the last line of content.

When to use

  • Mobile-first apps with 3–5 primary destinations
  • Anything that wants thumb-reachable navigation on phones
