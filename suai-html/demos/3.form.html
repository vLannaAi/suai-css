Form Element Summary

This page demonstrates all HTML form elements styled with the SUAI pla theme (Material Design 3).

Key Principles:

  • Semantic HTML: Use <small> for helper text, <strong> for error messages - no classes needed
  • Validation: Leverage HTML5 attributes (required, pattern, min/max) with CSS-only visual feedback
  • Accessibility: Proper <label> associations, fieldset grouping, and ARIA support
  • Material Design 3: Filled text field pattern, elevation shadows, semantic color system
  • Progressive Enhancement: Works without JavaScript, enhanced with CSS features

Part of the SUAI (Semantic Utility Adaptive Interface) framework
